# Packages to download
* npx create-expo-app AllergyAlarm
* cd AllergyAlarm
* npx expo install firebase
* npx expo install expo-router
* npm install @react-native-async-storage/async-storage
* npm install -g expo-cli
## To run application
* npx expo start
